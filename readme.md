# ColorRegressionApp

Веб-приложение для построения регрессионных моделей и прогнозирования параметров смесей на основе экспериментальных данных.

## Возможности

- Регистрация и авторизация пользователей;

- Ввод экспериментальных данных (X1, X2, X3, L, A, B);

- Построение регрессионных моделей;

- Расчёт коэффициентов регрессии для координат цвета L, A и B;

- Сохранение моделей в базе данных;

- Просмотр списка построенных моделей;

- Отображение формул регрессии и графиков сравнения;

- Прогнозирование параметров смеси по заданным значениям цвета;

- Хранение результатов прогнозирования.

## Используемые технологии

### Backend

- ASP.NET Core

- Entity Framework Core

- SQLite

- MathNet.Numerics

### Frontend

- HTML5

- CSS3

- JavaScript

- Chart.js


## Структура данных

Система использует базу данных SQLite для хранения:

- пользователей;

- регрессионных моделей;

- экспериментальных точек;

- коэффициентов регрессии;

- результатов прогнозирования.

## Запуск проекта

1. Перейти в каталог проекта:
cd ColorRegressionApp

2. Выполнить сборку:
dotnet build

3. Запустить приложение:
dotnet run

После запуска приложение будет доступно по адресу, указанному в консоли ASP.NET Core.

## Назначение

Приложение предназначено для автоматизации построения регрессионных моделей и прогнозирования параметров смесей на основе экспериментальных данных в задачах анализа и управления технологическими процессами.
